# Music

Make music. Play melodies and tones.

## Melodies

```cards
music.baDing.play()
music.baDing.playUntilDone()
music.baDing.loop()
music.baDing.stop()
```

## Tones and Music

```cards
music.playTone(Note.C, BeatFraction.Half)
music.ringTone(Note.C)
music.rest(BeatFraction.Half)
music.noteFrequency(Note.C)
music.changeTempoBy(20)
music.setTempo(120)
music.tempo()
music.beat()
music.setVolume(128)
```
